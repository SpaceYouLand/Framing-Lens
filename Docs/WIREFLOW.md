# Wireflow

## Primary flow (session-first) - (Default settings)

1. Landing
2. Session Setup
3. Prompt (repeat)
4. Results
5. Shared Results (if enabled by policy)

## Optional organizer (not required) - (Default settings)

- Workspace (one way to organize sessions in some environments)
- Codes/Invites (one way to collect sessions in a cohort)

## Screens (MVP) - (Default settings)

### 1) Landing - (Default settings)

- Start session

### 2) Session Setup - (Default settings)

- Select prompt set
- Select output modules (core + optional)
- Sharing policy selection (options depend on environment)
- Identity mode selection (options depend on environment)
