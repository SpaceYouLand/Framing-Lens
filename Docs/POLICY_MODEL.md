# Policy Model - (Default settings)

Behavior is determined by policy so the same tool can fit different environments.

## SessionPolicy (per session)- (Default settings) 

- **share_mode:** "private" | "link_view" | "public" | "collaborative"
- **identity_mode:** "anonymous" | "named_optional" | "named_required"
- **export_mode:** "off" | "on"
- **retention_mode:** "indefinite" | "days:N" | "environment_default"

**Notes:**
- Some environments prioritize privacy.
- Some environments prioritize openness, publishing, or collaboration.
- Policy exists to reflect environment intent.
